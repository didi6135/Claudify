project planning, see [`.planning/`](../.planning/).

## Style

- Second person ("you SSH to your server"), active voice
- Show the command first, explain after
- One example beats three paragraphs of theory
- If you find yourself writing "obviously," cut the sentence — it's not
  obvious to a new user
